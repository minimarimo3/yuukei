あなたはUI/UXデザインに精通したフロントエンドエンジニアです。
現在、UnityのuGUI向けUI素材を自動生成するWebツールの「新しいカラーテーマ」を作成しています。

システムは完全に分離されたアーキテクチャを採用しており、レイアウトやDOM構造を変更することはできません。
あなたのタスクは、指定された【テーマのコンセプト】に基づき、以下の `:root` 内のCSSカスタムプロパティ（配色）を上書きした新しいCSSコードと、リストに追加するためのJSONスニペットを生成することです。

【テーマのコンセプト】
・テーマ名: [例：サイバーパンク、ダークマター、抹茶、など]
・出力ファイル名: [例：cyberpunk.css]
・キーカラーや世界観: [例：ネオングリーンと漆黒。発光感を強めにして、文字色は視認性の高い白に。]

【上書きする変数のテンプレート】
/* 以下の変数のカラーコードのみを変更し、出力してください。要素の追加や削除は不要です。 */
:root {
  --hi: #ffffff;            /* 立体枠のハイライト（明るい色） */
  --lo: #7fe6ff;            /* 立体枠のシャドウ・アクセントカラー（濃い色） */
  --win-bg: #f7fbff;        /* ウィンドウの背景色 */
  --shadow: rgba(110, 210, 255, 0.22); /* 発光シャドウ・背景パターンの色 */
  --txt: #0a4a66;           /* 基本テキスト色（背景に対して読みやすい色） */
  --dot: rgba(90, 200, 255, 0.8);      /* 吹き出しやコンテンツ枠のドット線の色 */
  --title-a: #7fe6ff;       /* タイトルバーのグラデーション開始色 */
  --title-b: #e7f6ff;       /* タイトルバーのグラデーション終了色 */
  --title-shadow: rgba(255, 255, 255, 0.7); /* タイトル文字のドロップシャドウ */
  --btn-bg: #b9f2ff;        /* ボタンの通常時の背景色 */
  --btn-hover: #e7f6ff;     /* ボタンのホバー時の背景色（通常より少し明るく） */
  --btn-active: #7fe6ff;    /* ボタンの押し下げ時の背景色（通常より濃く） */
}

【出力要件（厳守）】
1. HTMLやJavaScriptのコードは一切出力しないでください。
2. 完成した `[出力ファイル名]` の中身となるCSSコードのブロックを出力してください。
3. `themes.json` に追記するための、以下の形式のJSONスニペットを出力してください。
   {
     "id": "[ファイル名から.cssを除いた文字列]",
     "name": "[テーマ名]",
     "file": "themes/[出力ファイル名]"
   }
