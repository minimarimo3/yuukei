# 台本（スクリプト）言語仕様書

> デスクトップマスコットの振る舞いを定義する「台本」の記述ルールおよび、ANTLR を用いたパーサー実装のための要件定義書です。

| [§1](01-基本ルール.md) | **基本ルール** | 全角・半角の正規化、スペース・改行・コメントの扱い、エンコーディング |
| [§2](02-台本の構造.md) | **台本の構造** | イベント・前提条件・初期値・シーンの階層構造、トリガー（合図・条件）の定義 |
| [§3](03-変数.md) | **変数（フラグ）** | 変数名の規則、予約語、型（真偽値・数値・文字列）、永続化と一時変数、算術演算・文字列結合 |
| [§4](04-条件式.md) | **条件式** | 一致・不一致、大小比較（後置形・中置形）、範囲比較、組み込み時間変数、時間範囲構文、複合条件 |
| [§5](05-関数呼び出し.md) | **関数呼び出し** | 位置引数・名前付き引数、出現可能な位置、セリフ内変数展開、戻り値 |
