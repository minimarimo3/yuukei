# 付録

## 付録A: 構文記号一覧

| 記号 | 用途 | 全角/半角 |
|------|------|----------|
| `##` | イベント宣言 | 半角（正規化後） |
| `###` | シーン宣言 | 半角（正規化後） |
| `=` | 代入、等価比較（後置形） | 半角（正規化後） |
| `!=` | 不一致比較 | 半角（正規化後） |
| `※` | 条件式の開始 | そのまま |
| `※それ以外` | Else 構文の開始 | そのまま（`それ以外` は予約語） |
| `（` `)` | 条件式・算術式のグループ化 | 全角・半角どちらも可 |
| `＜` `＞` | 関数呼び出し、変数展開（全角のみ） | 全角のみ |
| `<` `>` `<=` `>=` | 比較演算子（中置形、半角のみ） | 半角のみ |
| `==` `!=` | 等価・不一致比較（中置形） | 半角のみ |
| `「` `」` | セリフの囲み | そのまま |
| `→` | ジャンプ、イベント終了、シーン終了 | そのまま |
| `~` | 範囲演算子 | 半角（正規化後） |
| `＠` | システムイベント接頭辞 | 全角（正規化で統一） |
| `$$` | コメント開始 | 半角（正規化後） |
| `:` | ブロック開始、セクション宣言 | 半角（正規化後） |
| `_` | 一時変数接頭辞 | 半角（正規化後） |
| `+` `-` `*` `/` `%` | 算術演算子（`+` は文字列結合兼用） | 半角強制 |

## 付録C: 文の種類と記述ルール早見表

| 文の種類 | 1行に複数 | 条件付き1行記法 | ブロック内 |
|----------|----------|---------------|----------|
| セリフ（`「」`） | 可（表示列として） | 可（1つのみ） | 可 |
| 関数呼び出し（`＜＞`） | 可（表示列として） | 可（1つのみ） | 可 |
| セリフ＋関数の混合 | 可（表示列として） | 不可 | 可 |
| 代入（`変数=値`） | 不可 | 可（1つのみ） | 可 |
| ジャンプ（`→`） | 不可 | 可（1つのみ） | 可 |
| ブロック開始（`※（条件）:`） | 不可 | — | 可（ネスト） |
