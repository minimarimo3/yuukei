# 04 対話・アニメーション

## 1. 概要

キャラクターがセリフを「話す」演出を担うサブシステムです。  
吹き出し UI の表示・リサイズ・位置決め、および口パクアニメーションを提供します。

## 2. SpeechBubbleController（吹き出し表示）

### 2.1 責務

テキストの設定・表示・非表示を担うシンプルなファサードです。  
内部の UI 要素（`TextMeshPro`、背景、テール）への参照を持ちます。

### 2.2 インターフェース

```csharp
public class SpeechBubbleController : MonoBehaviour
{
    [SerializeField] private TMPro.TextMeshProUGUI textLabel;
    [SerializeField] private GameObject            bubbleRoot;

    /// <summary>
    /// テキストをタイプライター風に1文字ずつ表示する。
    /// delayPerChar は 1文字あたりの表示時間（秒）。0の場合は即時表示。
    /// UniTask を返し、全文表示完了まで await 可能にする。
    /// </summary>
    public async UniTask ShowTextAsync(string text, float delayPerChar, CancellationToken ct);

    /// <summary>吹き出しを非表示にする。</summary>
    public void Hide();
}
```

## 3. SpeechBubbleAutoResizer（自動リサイズ）

### 3.1 責務

TextMeshPro のテキスト量に応じて吹き出し背景の高さを動的に調整します。

### 3.2 リサイズアルゴリズム

```
Update()
  1. textLabel.preferredHeight を取得
  2. min/max の範囲にクランプ
  3. backgroundRect.sizeDelta.y を更新
  4. テールの位置を背景サイズに比例してシフト
```

### 3.3 インターフェース

```csharp
public class SpeechBubbleAutoResizer : MonoBehaviour
{
    [SerializeField] private TMPro.TextMeshProUGUI textLabel;
    [SerializeField] private RectTransform         backgroundRect;
    [SerializeField] private RectTransform         tailRect;
    [SerializeField] private float                 minHeight = 60f;
    [SerializeField] private float                 maxHeight = 300f;
}
```

## 4. SpeechBubblePositioner（吹き出し位置決め）

### 4.1 責務

キャラクターの頭ボーン位置を追跡し、吹き出しを適切な位置に配置します。

### 4.2 配置アルゴリズム

```
LateUpdate()
  1. Animator.GetBoneTransform(HumanBodyBones.Head) でヘッドボーンを取得
  2. ヘッドボーンのワールド座標をスクリーン座標に変換
  3. スクリーン座標を UI キャンバスのローカル座標に変換
  4. キャラクター回避：吹き出しがキャラクターと重なる場合に左右にシフト
  5. 画面端クランプ：余白を考慮してキャンバス範囲内に収める
  6. テール方向：キャラクター位置に応じて Scale.x を反転
```

### 4.3 インターフェース

```csharp
public class SpeechBubblePositioner : MonoBehaviour
{
    [SerializeField] private CharacterManager characterManager;
    [SerializeField] private RectTransform    bubbleRect;
    [SerializeField] private RectTransform    tailRect;
    [SerializeField] private Canvas           targetCanvas;
    [SerializeField] private float            horizontalMargin = 20f;
    [SerializeField] private float            canvasPadding    = 10f;
}
```

## 5. VRMLipSync（口パク）

### 5.1 責務

テキスト文字列を母音列に変換し、VRM の表情（Expression）を時系列で駆動します。  
日本語テキストに特化した母音マッピングを提供します。

### 5.2 母音マッピング

```
VRM ExpressionKey  対応文字（例）
─────────────────────────────────
Aa               あ行、か行、さ行、た行（タ・チ・ツ以外）、な行 ...
Ih               い行、き行、し行、ち、に行 ...
Ou               う行、く行、す行、つ、ぬ行 ...
Ee               え行、け行、せ行、て行、ね行 ...
Oh               お行、こ行、そ行、と行、の行 ...
（沈黙）         ん、っ → weight -1 で口を閉じる
```

カタカナも同様にマッピングします。

### 5.3 再生処理

```
SpeakAsync(text, ct)
  foreach character in text:
    1. 文字を母音カテゴリに変換
    2. ExpressionKey.CreateCustom(vowelName) でキーを取得
    3. weight を 0 → 1 に Lerp（speed: 20x/秒）
    4. 0.08 秒 待機（1 文字の表示時間）
    5. 次の文字へ → 現在の weight を 0 に戻す
```

### 5.4 インターフェース

```csharp
public class VRMLipSync : MonoBehaviour
{
    [SerializeField] private CharacterManager characterManager;

    // 1 文字あたりの表示時間（秒）
    [SerializeField] private float charDuration   = 0.08f;
    // 母音ウェイトの補間速度
    [SerializeField] private float lerpSpeed      = 20f;

    /// <summary>
    /// テキストを 1 文字ずつ口パクで再生する。
    /// テキスト全体の再生が終わるまで await する。
    /// </summary>
    public async UniTask SpeakAsync(string text, float charDuration, CancellationToken ct);

    /// <summary>キャラクター変更時にキャッシュを無効化する。</summary>
    public void InvalidateCache();
}
```

**定数**

| 定数 | 値 | 意味 |
|------|----|------|
| `CharDuration` | `0.08f` 秒 | 1 文字の口パク時間 |
| `LerpSpeed` | `20f` | 母音ウェイトの補間速度 |

### 5.5 エラーハンドリング

- VRM がロードされていない場合、テキスト長 × `charDuration` の時間だけ待機して返ります。
- 対応する表情が VRM に存在しない場合は、例外を握り潰してスキップします。
