# 02 起動シーケンス

## 1. 概要

`PlatformBootstrapper` がシーン上の唯一の Composition Root として機能します。  
起動時に全依存関係を組み立て、7 フェーズの非同期ブートシーケンスを実行します。

## 2. PlatformBootstrapper の責務

```csharp
/// <summary>
/// Composition Root。アプリ起動時に依存関係を組み立て、ブートシーケンスを実行する。
/// </summary>
public class PlatformBootstrapper : MonoBehaviour
{
    // Inspector から注入
    [SerializeField] private SettingsUIManager   settingsUI;
    [SerializeField] private CharacterManager    characterManager;
    [SerializeField] private ObjectToBottomRight objectToBottomRight;
    [SerializeField] private TriggerManager      triggerManager;
    [SerializeField] private PackageManager      packageManager;
    [SerializeField] private DaihonScenarioManager scenarioManager;

    private IAppIntegration    _appIntegration;
    private ICredentialStorage _credentialStorage;

    private void Awake()
    {
        // インフラ実装をここでのみ生成（Pure DI の Composition Root）
        _appIntegration    = PlatformServiceFactory.CreateAppIntegration();
        _credentialStorage = PlatformServiceFactory.CreateCredentialStorage();

        // IAppContext を初期化して登録
        AppContext.Initialize(_appIntegration, _credentialStorage);
    }

    private void Start()
    {
        BootSequenceAsync().Forget();
    }

    private void Update()
    {
        _appIntegration?.Tick();
    }

    private void OnDestroy()
    {
        _appIntegration?.Dispose();
    }
}
```

## 3. ブートシーケンス（7 フェーズ）

```
Phase 0  描画の一時停止
Phase 1  設定のロード
Phase 2  パッケージ・テーマ・プラグインの復元
Phase 3  キャラクターのロード
Phase 4  ローカルスクリプトの読み込み
Phase 5  ウィンドウ配置の計算
Phase 6  OS 連携の初期化（トレイ・マルチモニター・タスクバー非表示）
Phase 7  描画の再開 → トリガー監視の開始
```

### Phase 0：描画の一時停止

ユーザーに未完成な状態を見せないため、初期化完了まで描画を停止します。

```csharp
// メインカメラの cullingMask をゼロにして 3D 描画を停止
mainCam.cullingMask = 0;

// Canvas を無効化して UI 描画を停止
// （RectTransform の計算は維持して、後のレイアウト計算に使用）
foreach (var canvas in canvases)
    canvas.enabled = false;
```

### Phase 1：設定のロード

`ConfigManager.Awake()` で同期的に完了済みのため、フェーズとして確認のみ行います。  
`ConfigManager` は `ScriptableObject` ではなく `MonoBehaviour` として実装し、  
Awake で `persistentDataPath/settings.json` を読み込みます。

### Phase 2：パッケージ・テーマ・プラグインの復元

```csharp
await packageManager.RestorePackagesAsync();
```

`RestorePackagesAsync` は以下を順に実行します：
1. `settings.json` に記録されたインストール済みパッケージを列挙
2. 各パッケージの `plugins/*.dll` を `PluginLoader` でロード
3. 各パッケージの `theme/` を `ThemeManager` で適用
4. 各パッケージの `*.daihon` を `DaihonScenarioManager` にプリロード

### Phase 3：キャラクターのロード

```csharp
await characterManager.RestoreCharacterAsync();
```

`settings.json` の `currentCharacterId` が示す `.vrm` ファイルを非同期で読み込みます。  
ファイルが存在しない場合はスキップし、キャラクターなし状態で起動します。

### Phase 4：ローカルスクリプトの読み込み

```csharp
scenarioManager.LoadLocalScripts();
```

`persistentDataPath/Daihon/` 内の `.daihon` / `.txt` ファイルをスキャンし、  
メタデータ（シーン名・シグナル・条件）をキャッシュします。

### Phase 5：ウィンドウ配置の計算

```csharp
// PostLateUpdate まで待機してメッシュ Bounds を確定させる
await UniTask.Yield(PlayerLoopTiming.PostLateUpdate);

foreach (var positioner in objectToBottomRightList)
    positioner.PositionAtBottomRight();
```

Bounds 計算には VRM メッシュの描画パスが必要なため、`PostLateUpdate` まで待ちます。

### Phase 6：OS 連携の初期化

```csharp
_appIntegration.InitializeTray(
    onSettingsRequested: () => settingsUI.ShowSettings(),
    onQuitRequested:     Application.Quit
);
_appIntegration.SetupMultiMonitor();

// UniWindowController の初期化完了後にタスクバー非表示
await UniTask.WaitUntil(() => Time.frameCount > 5);
await UniTask.Delay(100);
_appIntegration.HideFromTaskbar();
```

### Phase 7：描画の再開とトリガー監視の開始

```csharp
// 描画の再開
mainCam.cullingMask = originalCullingMask;
foreach (var canvas in canvases)
    canvas.enabled = true;

// トリガー監視開始
triggerManager.StartMonitoring();
```

## 4. 起動シーケンスのエラーハンドリング

| 状況 | 挙動 |
|------|------|
| VRM ファイルが見つからない | 警告ログを出し、キャラクターなし状態で続行 |
| パッケージのロード失敗 | 該当パッケージをスキップして続行 |
| OS 連携の初期化失敗 | 警告ログを出し、機能なし状態で続行 |
| `settings.json` が壊れている | デフォルト設定で起動し、修復を試みる |

起動シーケンス中に例外が発生した場合は `Debug.LogError` で記録し、  
可能な限り後続フェーズの実行を継続します。

## 5. ConfigManager の設計

起動時に 1 回だけ読み込み、変更があるたびに書き込む設計です。

```csharp
public class ConfigManager : MonoBehaviour
{
    public static ConfigManager Instance { get; private set; }

    public AppSettings Settings { get; private set; }

    // 設定変更イベント（購読者はここで変化を検知）
    public event Action OnSettingsChanged;

    private void Awake()
    {
        if (Instance != null && Instance != this) { Destroy(gameObject); return; }
        Instance = this;
        DontDestroyOnLoad(gameObject);
        LoadSettings();
    }

    public void SaveSettings() { /* JSON シリアライズ → persistentDataPath/settings.json */ }
    private void LoadSettings() { /* JSON デシリアライズ */ }
}
```

```csharp
[Serializable]
public class AppSettings
{
    public string              currentCharacterId;
    public List<CharacterData> savedCharacters;
    public string              activePackageId;
    public List<PackageInfo>   installedPackages;
    public LLMSettings         llmSettings;
}

[Serializable]
public class CharacterData
{
    public string id;
    public string name;
    public string filePath;
}

[Serializable]
public class PackageInfo
{
    public string id;
    public string name;
    public string version;
    public string installPath;
    public bool   enabled;
}

[Serializable]
public class LLMSettings
{
    public string provider;    // "none" | "cloud" | "local"
    public string endpointUrl;
    public string modelName;
    // API キーは ICredentialStorage で管理（settings.json には保存しない）
}
```
